JavaScript Basics

1. Introduction to JavaScript

JavaScript is a high-level, interpreted programming language that is primarily used for web development. It is one of the core technologies of the World Wide Web, along with HTML and CSS.

2. Variables and Data Types

In JavaScript, you can declare variables using the var, let, or const keywords. JavaScript has several data types, including numbers, strings, booleans, and objects.

let name = "John Doe";
const age = 30;
let isStudent = true;

3. Operators and Expressions

JavaScript supports various operators, such as arithmetic, assignment, comparison, and logical operators. Expressions are combinations of values, variables, and operators that evaluate to a single value.

let sum = 5 + 3;
let isAdult = age >= 18;
let message = "Hello, " + name + "!";

4. Control Flow and Conditional Statements

JavaScript uses control flow structures like if-else statements, switch statements, and loops (such as for, while, and do-while) to control the execution of code.

if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Functions

1. Function Declarations and Expressions

Functions in JavaScript can be declared using the function keyword or assigned to a variable as a function expression.

function greet(name) {
    return "Hello, " + name + "!";
}

const calculateArea = function(width, height) {
    return width * height;
};

2. Arrow Functions

ES6 (ECMAScript 2015) introduced a new syntax for defining functions called arrow functions, which provide a more concise way to write functions.

const greet = (name) => {
    return "Hello, " + name + "!";
};

const calculateArea = (width, height) => width * height;

3. Scope and Closures

JavaScript has function scope, block scope, and global scope. Closures are a powerful feature that allow functions to access variables from an outer function, even after the outer function has finished executing.

function outerFunction() {
    const x = 5;

    function innerFunction() {
        const y = 10;
        console.log(x + y); // Can access x from the outer function
    }

    return innerFunction;
}

const myInnerFunction = outerFunction();
myInnerFunction(); // Output: 15

DOM Manipulation

1. Selecting and Accessing DOM Elements

JavaScript allows you to select and access elements in the Document Object Model (DOM) using methods like document.getElementById(), document.querySelector(), and document.querySelectorAll().

const heading = document.getElementById("main-heading");
const paragraphs = document.querySelectorAll("p");
const firstParagraph = document.querySelector("p");

2. Modifying DOM Elements

Once you have selected DOM elements, you can manipulate their properties, such as textContent, innerHTML, style, and more.

heading.textContent = "New Heading";
paragraphs[0].style.color = "red";
firstParagraph.innerHTML = "<em>This is a paragraph</em>";

3. Event Handling

JavaScript enables you to listen for and respond to user events, such as clicks, mouse movements, and key presses, using event listeners and event handlers.

const button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("Button clicked!");
});

Asynchronous JavaScript

1. Callbacks

Callbacks are functions that are passed as arguments to other functions and are called when a certain event occurs or a certain task is completed.

function fetchData(callback) {
    // Simulate asynchronous data fetching
    setTimeout(function() {
        const data = "Fetched data";
        callback(data);
    }, 2000);
}

fetchData(function(data) {
    console.log(data);
});

2. Promises

Promises provide a more structured way to handle asynchronous operations. They represent the eventual completion (or failure) of an asynchronous operation and its resulting value.

function fetchData() {
    return new Promise((resolve, reject) => {
        // Simulate asynchronous data fetching
        setTimeout(() => {
            const data = "Fetched data";
            resolve(data);
        }, 2000);
    });
}

fetchData()
    .then(data => console.log(data))
    .catch(error => console.error(error));

3. Async/Await

Async/await is a syntactical sugar on top of promises, making asynchronous code easier to write and read. The async keyword is used to define an asynchronous function, and the await keyword is used to wait for a promise to resolve.

async function fetchData() {
    try {
        const response = await fetch("https://api.example.com/data");
        const data = await response.text();
        console.log(data);
    } catch (error) {
        console.error("Error fetching data:", error);
    }
}

fetchData();

Modern JavaScript Features

1. ES6 (ECMAScript 2015)

ES6 introduced many new features to JavaScript, such as let and const for improved variable declarations, arrow functions, classes, modules, and more.

// ES6 Arrow Function
const square = (x) => x * x;

// ES6 Class
class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log(Hello, my name is ${this.name}.);
    }
}

const john = new Person("John", 30);
john.greet();

2. Destructuring

Destructuring is a way to extract values from arrays or properties from objects and assign them to variables in a more concise way.

const person = { name: "John", age: 30 };
const { name, age } = person;
console.log(name); // Output: "John"
console.log(age); // Output: 30

const numbers = [1, 2, 3];
const [first, second] = numbers;
console.log(first); // Output: 1
console.log(second); // Output: 2

3. Modules

Modules in JavaScript allow you to organize your code into reusable and maintainable units. They help manage dependencies and promote encapsulation.

// math.js
export const add = (a, b) => a + b;
export const subtract = (a, b) => a - b;

// app.js
import { add, subtract } from './math.js';

console.log(add(2, 3)); // Output: 5
console.log(subtract(5, 3)); // Output: 2

4. Spread and Rest Operators

The spread operator (...) allows you to spread the elements of an array or the properties of an object. The rest operator (...) collects multiple elements into an array.

// Spread Operator
const numbers = [1, 2, 3];
const newNumbers = [...numbers, 4, 5];
console.log(newNumbers); // Output: [1, 2, 3, 4, 5]

// Rest Operator
function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}
console.log(sum(1, 2, 3)); // Output: 6
